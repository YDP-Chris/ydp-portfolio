## What it does

PulseCheck is a batch job heartbeat monitor for Microsoft Dynamics 365 that transforms you from reactive to proactive. No more discovering failed jobs from angry users â€“ catch issues before they impact your business.

**The Chris Story**: It's Monday morning. Chris opens PulseCheck and immediately sees the customer import job showing 'Time since last success: 14h ago' in bold red. He fixes the issue before a single user notices. For the first time in months, he's ahead of the problems instead of chasing them.

### Key Features
- **Real-time job status** - See all your D365 batch jobs at a glance
- **Time since last success** - Know exactly how long jobs have been failing
- **Visual alerts** - Bold red warnings for jobs that need attention
- **Heartbeat monitoring** - Continuous health checks for critical processes

## How to use it

1. **Connect to D365** - Link PulseCheck to your Dynamics 365 environment
2. **Configure monitoring** - Select which batch jobs to track
3. **Set thresholds** - Define what constitutes a "failed" or "overdue" job
4. **Stay informed** - Check the dashboard regularly or set up notifications

### Quick Start
```
1. Install PulseCheck in your D365 environment
2. Navigate to System Administration > PulseCheck
3. Add your critical batch jobs to monitoring
4. Customize alert thresholds
5. Bookmark the dashboard for daily checks
```
